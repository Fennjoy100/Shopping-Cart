Shopping Cart E-Commerce Website MERN Stack

This is a full stack Shopping Cart E-Commerce web application built using the MERN stack MongoDB Express.js React.js and Node.js. The application allows users to browse products add items to the cart update quantities remove items and manage their shopping experience through a simple and user friendly interface.

Features
User friendly product listing
Add to cart functionality
Update product quantity in cart
Remove items from cart
Dynamic cart total calculation
Responsive design

Tech Stack
Frontend
React.js
CSS or Bootstrap

Backend
Node.js
Express.js

Database
MongoDB

Folder Structure
client - React frontend
server - Node.js and Express backend

Installation and Setup
Clone the repository
git clone https://github.com/your-username/your-repo-name.git

Go to project folder
cd your-repo-name

Install backend dependencies
cd server
npm install

Install frontend dependencies
cd client
npm install

Run the application

Start backend
cd server
npm start

Start frontend
cd client
npm start

Open in browser
http://localhost:3000
